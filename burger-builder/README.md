## About The Project

A burger builder website where user can build their own burger and place the order online. They also can login and see their order history.

This project is based on the [React Course](https://www.udemy.com/course/react-the-complete-guide-incl-redux/) on Udemy. Great thanks to the instructor [Maximilian Schwarzm√ºller](https://www.udemy.com/user/maximilian-schwarzmuller/).

### Built With

- [React](https://reactjs.org/) : components
- [Redux](https://redux.js.org/) : state management
- [React Router](https://reactrouter.com/) : routing
- [axios](https://github.com/axios/axios) : HTTP request and response
- [firebase](https://firebase.google.com/) : database and authentication
